这是一款教务管理系统，
在此系统中，你可以以学生、教师、教务管理人员三种身份进行注册。
成功注册后，系统会记录你的信息。
成功注册并且登录系统后，你会进入到不同身份的相应界面，并且可以使用相应的功能。
以学生身份注册，登录系统后，
你可以查看当前课表，对已修课程进行成绩查询，还可以进行选课操作。
在选课操作中，你需要先查看可选课程，然后再去更改自己的课表。
以教师身份注册，登录系统后，
你可以查看自己的授课课表，课程的学生名单，学生对你的课程的评价。
最后，你也可以为你的学生进行课程打分。
以教务管理人员身份注册，登录系统后，
你可以向课程库增加、删减课程，并且查看当前可选课程。
部分功能还有待完善，敬请谅解！
最后，祝你使用愉快！
